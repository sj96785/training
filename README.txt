重訓課表紀錄 PWA

一、目前已完成功能
1. 可安裝為 PWA。
2. 可離線使用。
3. 內建上肢A（主推）、上肢B（主拉）、腿日模板。
4. 已依照你目前的一週三練邏輯設定：前兩動作各 3 組休 180 秒，後兩動作各 2 組休 150 秒。
5. 可自行新增、編輯模板與動作。
6. 可開始訓練並逐組記錄重量、次數、RPE、熱身、完成狀態。
7. 可在訓練中新增動作、補入模板動作、刪除組數、調整動作順序。
8. 可查看歷史紀錄。
9. 可查看最近 7 天、30 天統計，以及各動作完整統計。
10. 可匯出 / 匯入 JSON 備份。
11. 開始訓練或新增動作時，若有歷史紀錄會自動帶入上次正式組。
12. 訓練中畫面已改成更適合手機點按的卡片式輸入。
13. 舊版內建的推日、拉日、推拉日會自動替換成你現在的上肢A / 上肢B / 腿預設。

二、目前內建模板
1. 上肢A（主推）
   - 啞鈴肩推：3 組，休 180 秒
   - 槓鈴划船：3 組，休 180 秒
   - 上斜啞鈴臥推：2 組，休 150 秒
   - 引體向上：2 組，休 150 秒
2. 上肢B（主拉）
   - 負重引體向上：3 組，休 180 秒
   - 倒立伏地挺身：3 組，休 180 秒
   - 反手引體向上：2 組，休 150 秒
   - 負重伏地挺身：2 組，休 150 秒
3. 腿日
   - 頸前深蹲：3 組，休 180 秒
   - 分腿蹲（雙腳互換）：3 組，休 180 秒
   - 器械股四頭：2 組，休 150 秒
   - 器械股二頭：2 組，休 150 秒

三、使用方式
1. 將整個資料夾上傳到 GitHub Pages，或用本機伺服器開啟。
2. 第一次進入後即可開始使用。
3. 手機瀏覽器若支援，可安裝到桌面。

四、注意事項
1. 第一次載入時需要連網，之後才可離線。
2. 建議定期在「設定」中匯出 JSON 備份。
3. 若你要我下一版再加：
   - 超級組 / 遞減組
   - 體重、圍度、照片追蹤
   - 每週訓練天數規劃
   - PR 提醒
   - 更細的健美分化模板
   我可以直接接著改。
