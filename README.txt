重訓課表紀錄 PWA

一、目前已完成功能
1. 可安裝為 PWA。
2. 可離線使用。
3. 內建推日、拉日、腿日、推拉日（目前常用）模板。
4. 可自行新增、編輯模板與動作。
5. 可開始訓練並逐組記錄重量、次數、RPE、熱身、完成狀態。
6. 可在訓練中新增動作、補入模板動作、刪除組數、調整動作順序。
7. 可查看歷史紀錄。
8. 可查看最近 7 天、30 天統計，以及各動作完整統計。
9. 可匯出 / 匯入 JSON 備份。
10. 推拉同日已優先排序，首頁與訓練頁會先推薦。
11. 開始訓練或新增動作時，若有歷史紀錄會自動帶入上次正式組。
12. 訓練中畫面已改成更適合手機點按的卡片式輸入。

二、使用方式
1. 將整個資料夾上傳到 GitHub Pages，或用本機伺服器開啟。
2. 第一次進入後即可開始使用。
3. 手機瀏覽器若支援，可安裝到桌面。

三、注意事項
1. 第一次載入時需要連網，之後才可離線。
2. 建議定期在「設定」中匯出 JSON 備份。
3. 若你要我下一版再加：
   - 超級組 / 遞減組
   - 體重、圍度、照片追蹤
   - 每週訓練天數規劃
   - PR 提醒
   - 更細的健美分化模板
   我可以直接接著改。
